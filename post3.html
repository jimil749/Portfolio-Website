Personalised Expense Manager

Github Link

I worked on this project in July 2020, and it took me about a month to complete. Expense Manager is a full stack web app, designed to let customers manage their expenses efficiently.

Allows users to categorize and track their daily expenses, with categories such as Food, shopping, leisure, groceries etc.

Provides users with intuitive bar-charts and pie charts based on their expenses in a given time frame, which allows users to get more understanding of the expenses.

Monthly reports enable users to get more insight into their expenses and help them save money.

Technologies:
  • - NodeJS
  • - Express
  • - Victory JS
  • - ReactJS
  • - Material UI
  • - MongoDB
  • - Heroku
